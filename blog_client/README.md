Technologies used:
  - React.js, react-hook-form, react-toastify
  - Redux-Toolkit, redux-persist
  - Typescript
  - CSS(SCSS)
  - Axios

# Run project
- clone the repo
- npm install
- npm start

# Available to use
Auth:
- Register form
- Login form
Posts:
- Homepage with posts
- Add post from modal window
- Delete post
Personal Info:
- Update info
